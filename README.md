# es7-express

# Getting started
- Ensure that you have Docker installed


# Developing
- Run `$ npm run build`
- Run `$ npm run serve:dev`
- TODO: Add live updates in Docker container

# Serving
- Run `$ docker-compose build && docker-compose up`
- Docker will serve endpoints at http://0.0.0.0:8080

# Testing
- Run `$ npm test`
